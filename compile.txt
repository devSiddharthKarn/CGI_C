lib src compilation is important as it is important since the lib is still in development

compile from the main folder


with windows build binary
gcc test/test.c -Lbuild/windows -lcgi -lgdi32 -o test/testBinaryBuild.exe

for window win32 lgdi32
gcc test/test.c src/main.c -o test.exe -lgdi32  -Iinclude


with Linux build binary
 gcc test/test.c -o test/testLinuxBinaryBuild.out -Lbuild/linux -lcgi -lX11 -lXrandr

for linux based on X11
gcc test/test.c src/main.c -lX11 -lXext -lXrandr -Iinclude -o test/test.out -lm
